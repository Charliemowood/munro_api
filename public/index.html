Welcome to the Munro API

This is a work in progress, with more data & routes to be added.

The current routes available are:

  • https://munroapi.herokuapp.com/api/munros - View all Munros
  • https://munroapi.herokuapp.com/api/munros/name/:name - View Munros by Name
  • https://munroapi.herokuapp.com/api/munros/smcid/:smcid - View Munros by SMC ID
  • https://munroapi.herokuapp.com/api/munros/region/:region - View Munros by Region

No API key is required.
